Installation Instructions¶

Obtaining the Latest Software¶

The latest version of the Time Encoding and Decoding Toolbox can be downloaded from the Bionet Code Repository.

Prerequisites¶

The Python implementation of the Time Encoding and Decoding Toolbox requires that several software packages be present in order to be built and installed (older versions of these packages may work, but have not been tested):

  • Python 2.5 or later.
  • Cython 0.11.2 or later.
  • Numpy 1.2.0 or later.
  • Scipy 0.7.0 or later.
  • Bionet Python Utilities 0.015 or later.

To run the CUDA-dependent implementations, you will also need

  • NVIDIA CUDA Toolkit 3.0 or later.
  • PyCUDA 0.94rc or later.
  • scikit.cuda 0.01 or later.

To run the demo code and generate plots, the following package is also required:

  • Matplotlib 0.98 or later.

To build the documentation, the following packages are also required:

  • Docutils 0.5 or later.
  • Sphinx 1.0.1 or later.
  • Jinja2 2.2 or later
  • Pygments 0.8 or later

This software has been tested on Linux; it should also work on other platforms supported by the above packages.

Building and Installation¶

To build and install the toolbox, download and unpack the source release and run:

python setup.py install

from within the main directory in the release. Sample code demonstrating how to use the toolbox is located in the demos/ subdirectory.

To rebuild the documentation, run:

make

from within the doc/ subdirectory and follow the directions.
