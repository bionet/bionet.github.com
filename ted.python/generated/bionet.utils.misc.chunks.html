bionet.utils.misc.chunks¶

bionet.utils.misc.chunks(seq, n)¶

Chunk generator.

Return a generator whose next() method returns length n subsequences of the given sequence. If len(seq) % n != 0, the last subsequence returned will contain fewer than n entries.

Parameters :

seq : iterable

Sequence to split into chunks.

n : int

Chunk size.
Returns :

g : generator

Generator that will return length n chunks of seq.
